## 🌟 重大更新

### 1. 全新首页 - 功能选择模块化 ⭐⭐⭐⭐⭐

**不再直接进入拍照页面！**

**新首页设计**：
- 🎨 渐变背景 + 大卡片设计
- 📷 拍照翻译卡片："拍照识别 · 框选区域 · 精准翻译"
- ✍️ 文字翻译卡片："输入文字 · 快速翻译 · 支持多语言"
- 🎤 语音翻译卡片："语音输入 · 实时识别 · 即说即译"

**交互流程**：
```
启动App → SplashActivity（小冷在这 ❄️）
         ↓

         └─ 点击"语音翻译" → 语音识别页面
```

**底部导航**：
- 🏠 首页（功能选择）
- 📚 历史记录

---

### 2. 语音翻译功能 ⭐⭐⭐⭐⭐

**全新功能模块！**

**使用方式**：
1. 从首页点击"语音翻译"卡片
2. 点击麦克风图标开始说话
3. 实时显示识别的文字
4. 自动翻译成中文
5. 可复制、分享翻译结果

**技术特性**：
- 使用 Android SpeechRecognizer API
- 支持中文语音识别
- 实时显示部分识别结果
- 友好的错误提示
- 录音状态可视化

**权限要求**：
- 首次使用需授予录音权限（RECORD_AUDIO）

**适用场景**：
- 口译需求
- 快速记录翻译
- 不方便打字时

---

### 3. 拍照后区域选择 ⭐⭐⭐⭐⭐

**解决整面翻译的痛点！**

**新的拍照流程**：
```
拍照/选图 → 图片裁剪页面（框选区域）→ 确认 → OCR + 翻译
```

**裁剪页面功能**：
- ✂️ 可拖动的矩形选择框
- 🎯 拖动四个角调整大小
- ↔️ 拖动四条边调整位置
- 🖐️ 拖动中心整体移动
- 📐 九宫格辅助线帮助构图
- 🌑 半透明遮罩显示选择外区域

**交互细节**：
- 白色边框 + 圆形角点标记
- 顶部工具栏：取消 / 确认
- 底部提示："拖动边框调整翻译区域"
- 最小选择框尺寸限制

**优势**：
- ✅ 只翻译关心的区域
- ✅ 避免无关文字干扰
- ✅ 提高OCR准确率
- ✅ 节省翻译时间

## 📝 使用指南

### 拍照翻译（带区域选择）
1. 从首页点击"拍照翻译"卡片
2. 点击大相机按钮拍照
3. **新功能**：进入裁剪页面，拖动边框选择翻译区域
4. 点击"确认"
5. 小冷自动识别选中区域的文字并翻译
6. 可分享或查看历史

### 语音翻译
1. 从首页点击"语音翻译"卡片
2. 点击麦克风图标
3. 授予录音权限（首次）
4. 开始说话
5. 实时看到识别文字
6. 自动翻译成中文
7. 可复制或分享结果

### 文字翻译
1. 从首页点击"文字翻译"卡片
2. 输入要翻译的文字
3. 点击"让小冷来翻译"
4. 可复制或分享结果

### 查看历史
1. 点击底部"历史记录"标签
2. 查看最近5条翻译
3. 点击"清空"可删除全部历史

## 🔒 隐私保护

**新增权限**：
- RECORD_AUDIO（录音权限）- 仅语音翻译使用

**隐私承诺**：
- ✅ 语音识别：设备端处理（Android API）
- ✅ OCR识别：设备端处理（ML Kit）
- ✅ 翻译：设备端处理（TFLite/ML Kit）
- ✅ 图片裁剪：本地处理
- ✅ 历史记录：仅本地存储
- ✅ 零数据上传：不连接任何后端

## 📦 安装说明

### 权限要求
