### 2. Plan C: 混合翻译方案 ⭐⭐⭐⭐⭐

**核心创新**：首创混合翻译架构，解决模型下载痛点

**问题背景**：
- ML Kit 首次需下载 ~30MB 模型
- 用户需要等待下载才能使用
- 网络不佳时体验糟糕

**解决方案**：
```
TFLite 本地模型 ───┐
                   ├─→ HybridTranslator ─→ 智能切换
ML Kit 在线模型 ───┘
```

**翻译策略**：
1. 首次启动：立即可用 TFLite 本地模型（离线翻译）
2. 后台下载：静默尝试下载 ML Kit 模型
3. 自动升级：下载成功后自动切换到高质量翻译
4. 降级保护：ML Kit 失败时自动降级到 TFLite

**用户体验**：
- ✅ 启动即用，零等待
- ✅ 完全离线可用
- ✅ 自动在线增强
- ✅ 显示翻译方法：💡 在线翻译 / 💡 离线翻译

**技术实现**：
- `TFLiteTranslator`: 预打包轻量模型（当前为词典映射占位）
- `HybridTranslator`: 智能切换逻辑 + 降级保护
- `TranslateViewModel`: 协程初始化 + 后台升级

### 3. 从相册选择图片翻译 ⭐⭐⭐⭐⭐

**功能说明**：
- 不再局限于拍照，可以选择已有图片
- 适用场景：
  - 截图翻译
  - 旧照片文字识别
  - 从其他App分享的图片

**使用方式**：
- 相机页面：点击相册按钮（📷 旁边的小按钮）
- 选择图片 → 自动OCR → 自动翻译

**技术细节**：
- Activity Result API: `PickVisualMedia`
- 兼容新旧版本：ImageDecoder (Android P+) / MediaStore (旧版)
- URI 转 Bitmap 后复用现有翻译流程

## 📝 使用指南

### 拍照翻译
1. 打开App，默认进入拍照页面
2. 点击大相机按钮拍照
3. 小冷自动识别文字并翻译
4. 可分享或查看历史

### 相册翻译
1. 在拍照页面，点击小相册按钮
2. 选择图片
3. 自动OCR + 翻译

### 文字翻译
1. 切换到"文字翻译"标签
2. 输入要翻译的文字
3. 点击"让小冷来翻译"
4. 可复制或分享结果

### 查看历史
1. 切换到"历史记录"标签
2. 查看最近5条翻译
3. 点击"清空"可删除全部历史

## 🔒 隐私保护

**零数据上传**：
- ✅ OCR识别：设备端处理（ML Kit）
- ✅ 翻译：设备端处理（TFLite本地模型 或 ML Kit设备端模型）
- ✅ 历史记录：仅存储在本地（SharedPreferences）
- ✅ 无服务器：不连接任何后端服务器
- ✅ 无埋点：不收集任何用户数据

**唯一网络请求**：
- ML Kit 首次下载翻译模型（可选，约30MB）
- 用户可选择仅使用离线TFLite模型

## 📦 安装说明

### 方式1: 下载APK
1. 前往 [GitHub Releases](https://github.com/LYP661/translator/releases)
2. 下载 `app-release.apk`
3. 安装到Android手机

### 方式2: 自行编译
```bash
git clone https://github.com/LYP661/translator.git
cd translator
./gradlew assembleRelease
```

**系统要求**：
- Android 8.0+ (API 26+)
- 建议Android 9.0+ 以获得最佳体验
